def read_entries():
   if not os.path.exists(DATA_FILE):
     return []
  with open(DATA_FILE, 'r', encoding='utf-8') as f:

        return json.load(f)
     except json.JSONDecodeError:
       return []

This function:

  • Checks if entries.json exists
  • Opens it safely and converts JSON text ‚Üí Python list
  • Returns [] if file is empty or corrupted

  • Uses indent=2 for nice formatting

GETFetch Entries Route:

@app.route('/api/entries', methods=['GET'])
def get_entries():
    entries = read_entries()
     return jsonify(entries)

This route:

  • @app.route('/api/entries'): Defines a URL endpoint
  • methods=['GET']: Responds to GET requests (used to read data)
  • Returns the list of all entries as a JSON response

POSTAdd New Entry Route:

@app.route('/api/entries', methods=['POST'])
def add_entry():
     data = request.get_json()
     name = data.get('name', 'Anonymous').strip()
     message = data.get('message', '').strip()

     if not message:
         return jsonify({'error': 'Message is required'}), 400

     new_entry = {
          'id': int(datetime.now().timestamp() * 1000),
          'name': name,
          'message': message,
          'createdAt': datetime.now().isoformat()
          }

     entries = read_entries()
     entries.insert(0, new_entry)
     write_entries(entries)

     return jsonify(new_entry), 201

Step by step:

  • Reads data from frontend using request.get_json()
  • .get('name', 'Anonymous'): If name is missing, defaults to ‚ÄúAnonymous.‚Äù
  • .strip(): Removes leading/trailing spaces.
  • Checks if message is empty ‚Äî if so, returns an error HTTP 400 Bad Request

  • Creates a dictionary for the new guestbook entry.
  • Uses a timestamp datetime.now().timestamp() * 1000 as a unique ID.
  • Stores time in ISO format for display later.

  • Reads the current entries.
  • Inserts new one at the top (so newest appears first).
  • Writes it back to entries.json
  • Returns the newly added entry as confirmation with status code 201 Created

üè† Serving the Frontend

‚ñº
@app.route('/')
def serve_index():
    return send_from_directory('static', 'index.html')

@app.route('/')
def serve_static_file(path):
    return send_from_directory('static', path)

if __name__ == '__main__':
    app.run(debug=True, port=5000)

These routes handle:

  • The first route serves the main HTML page index.html
  • The second route serves any other file inside /static ‚Äî CSS, JS, images, etc.
  • This makes Flask act like a mini web server for your frontend files

  • debug=True enables auto-reloading and helpful error messages
  • Server runs on port 5000
